# 15. 三数之和

> 难度：**Medium** | 分类：双指针 | 标签：数组, 双指针, 排序

> 🔗 力扣链接：[https://leetcode.cn/problems/3sum/](https://leetcode.cn/problems/3sum/)

## 📝 题目描述

给你一个整数数组 `nums`，找出所有不重复的三元组 `[a, b, c]`，使得 `a + b + c = 0`。答案中不能有重复的三元组。

- 题号：15
- 题目名：三数之和
- 难度：中等
- 题型：双指针
- 为什么选这题：它是双指针夹逼的经典应用，也是去重逻辑的典型训练题。

## 🧠 思考路径

朴素想法是三重循环枚举所有三元组，检查和是否为 0。但这样是 `O(n³)`，而且去重很麻烦。

关键优化：先排序。排序后可以利用"有序"这个性质：固定一个数 `a`，然后在它右侧的区间里用双指针找 `b + c = -a`。因为数组有序，左指针太小就右移，右指针太大就左移，一次遍历就能找完所有组合。

去重的方法是：排序后相同的数会相邻，遇到和前一个相同的就跳过。

## 💻 Java 题解

### 解法 1：排序 + 双指针

排序数组。固定第一个数 nums[i]，在 i+1 到末尾用双指针找两数之和等于 -nums[i]。跳过重复值去重。

- ⏱ 时间复杂度：`O(n²)`
- 💾 空间复杂度：`O(1)`

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        Arrays.sort(nums);
        for (int i = 0; i < nums.length - 2; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) continue; // 去重
            if (nums[i] > 0) break; // 剪枝：最小的都>0
            int left = i + 1, right = nums.length - 1;
            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];
                if (sum == 0) {
                    result.add(Arrays.asList(nums[i], nums[left], nums[right]));
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    left++; right--;
                } else if (sum < 0) left++;
                else right--;
            }
        }
        return result;
    }
}
```

**关键点：**
- 排序是双指针的前提
- 三处去重：i、left、right
- nums[i]>0 提前剪枝

**执行步骤：**
1. 排序
1. 固定 i 从 0 遍历
1. left=i+1, right=n-1
1. sum<0 则 left++, sum>0 则 right--
1. sum==0 记录结果并跳过重复值


## ⚠️ 易错点

- 去重必须在三个位置都做：`i` 跳过重复的 `a`，`left` 跳过重复的 `b`，`right` 跳过重复的 `c`。
- 剪枝条件不要遗漏：如果当前 `a` 加上后面最小的两个数都大于 0，后面的 `a` 更大，直接 break。
- 找到一组解后，`left` 和 `right` 都要移动，不能只移一个，否则会死循环。

- 忘记去重导致重复结果

- 去重条件是 nums[i]==nums[i-1] 不是 nums[i]==nums[i+1]

## 📊 复杂度分析

- 时间复杂度：`O(n²)`。排序 `O(n log n)`，外层循环 `O(n)`，内层双指针 `O(n)`。
- 空间复杂度：`O(1)`（不考虑排序的额外空间和输出数组）。
