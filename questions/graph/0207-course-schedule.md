# 207. 课程表

> 难度：**Medium** | 分类：图论 | 标签：深度优先搜索, 广度优先搜索, 图, 拓扑排序

> 🔗 力扣链接：[https://leetcode.cn/problems/course-schedule/](https://leetcode.cn/problems/course-schedule/)

## 📝 题目描述

给你 `numCourses` 门课程和一个先修列表 `prerequisites`，其中 `prerequisites[i] = [a, b]` 表示要学课程 `a` 必须先学课程 `b`。判断能否学完所有课程。

重点不在"会不会排课表"，而在"怎么检测先修关系里有没有环"。

- 题号：207
- 题目名：课程表
- 难度：中等
- 题型：图论
- 为什么选这题：它是拓扑排序的入门题，能帮你建立"有向无环图 + 入度"的核心图论思维。

## 🧠 思考路径

这个问题本质上是：给一个有向图，判断其中有没有环。如果没有环（有向无环图，DAG），就能找到一个合法的学习顺序；如果有环（比如"学 A 要先学 B，学 B 要先学 A"），就永远学不完。

检测有向图是否有环，最经典的方法是拓扑排序。具体做法是用 BFS 配合"入度"：

1. 统计每个节点的入度（有多少条边指向它）。入度为 0 表示没有前置依赖，可以先学。
2. 把所有入度为 0 的节点放入队列。
3. 从队列中取出一个节点，把它指向的所有邻居的入度减 1。如果减到 0，说明它的前置条件都满足了，也放入队列。
4. 最后看处理了多少个节点。如果等于总课程数，说明没有环；如果有节点没被处理到，说明这些节点在环里。

也可以用 DFS 做拓扑排序，通过颜色标记（白/灰/黑）来检测环。

这题的关键词不是"排课表"，而是"拓扑排序检测有向图中的环"。

## 💻 Java 题解

### 解法 1：BFS 拓扑排序（Kahn算法）

建图+入度数组。入度为0的入队，弹出时减少邻居入度。最终已修课程数=总数则无环。

- ⏱ 时间复杂度：`O(V+E)`
- 💾 空间复杂度：`O(V+E)`

```java
class Solution {
    public boolean canFinish(int numCourses, int[][] prerequisites) {
        List<List<Integer>> graph = new ArrayList<>();
        int[] inDegree = new int[numCourses];
        for (int i = 0; i < numCourses; i++) graph.add(new ArrayList<>());
        for (int[] pre : prerequisites) {
            graph.get(pre[1]).add(pre[0]);
            inDegree[pre[0]]++;
        }
        Queue<Integer> q = new LinkedList<>();
        for (int i = 0; i < numCourses; i++)
            if (inDegree[i] == 0) q.offer(i);
        int count = 0;
        while (!q.isEmpty()) {
            int course = q.poll();
            count++;
            for (int next : graph.get(course))
                if (--inDegree[next] == 0) q.offer(next);
        }
        return count == numCourses;
    }
}
```

**关键点：**
- 入度为0先入队
- 弹出后邻居入度-1
- count==numCourses 表示无环

**执行步骤：**
1. 建图+入度数组
1. 入度0的入队
1. BFS弹出减少邻居入度
1. 判断已修课程数


## ⚠️ 易错点

- 先修关系 `[a, b]` 表示的是"先学 b 才能学 a"，建图时边是 `b → a`，方向别搞反。
- BFS 法中入度为 0 的课程可能不止一门，初始时都要入队。
- DFS 法中三态标记缺一不可：`0` 未访问、`1` 正在当前路径上（遇到说明有环）、`2` 已完成（无需再处理）。

- prerequisites[i] = [a,b] 表示先修b再修a

- count==numCourses 才说明无环

## 📊 复杂度分析

- 时间复杂度：`O(V + E)`。每个节点和每条边各访问一次。
- 空间复杂度：`O(V + E)`。邻接表存储图。
