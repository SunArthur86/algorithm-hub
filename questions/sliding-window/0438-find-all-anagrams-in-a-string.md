# 438. 找到字符串中所有字母异位词

> 难度：**Medium** | 分类：滑动窗口 | 标签：哈希表, 字符串, 滑动窗口

> 🔗 力扣链接：[https://leetcode.cn/problems/find-all-anagrams-in-a-string/](https://leetcode.cn/problems/find-all-anagrams-in-a-string/)

## 📝 题目描述

给你两个字符串 `s` 和 `p`，找出 `s` 中所有 `p` 的字母异位词的起始索引。字母异位词是指由相同字母以不同顺序排列组成的字符串。

重点不在"全排列"，而在"固定窗口内的字符频率是否匹配"——这是滑动窗口最典型的应用场景。

- 题号：438
- 题目名：找到字符串中所有字母异位词
- 难度：中等
- 题型：滑动窗口
- 为什么选这题：它是滑动窗口的教科书级题目，窗口大小固定，考察的是"如何用计数器判断两个字符串是否互为异位词"。

## 🧠 思考路径

暴力做法：对 `s` 的每个长度为 `len(p)` 的子串，判断它是否和 `p` 互为异位词。每次判断需要排序或计数，总复杂度 `O(n × m)`。

优化思路：窗口大小固定为 `len(p)`，窗口每次向右滑动一格。关键在于：窗口右移时，只有左边出一个字符、右边进一个字符，大部分字符的频率不变。所以我们可以维护一个字符频率计数器，每次滑动时只更新两个字符的计数，然后检查频率是否匹配。

具体做法：用两个计数数组（或哈希表）分别记录 `p` 的字符频率和当前窗口的字符频率。为了不用每次比较整个数组，可以维护一个 `match` 变量，记录当前有多少种字符的频率已经匹配。

更简洁的写法：只用一个 diff 数组，记录 `窗口频率 - p频率`。diff 全为 0 时就匹配了。用一个 `diffCount` 记录 diff 不为 0 的字符种数，避免每次遍历整个数组。

所以这题的关键词是"固定窗口 + 字符频率差值"。

## 💻 Java 题解

### 解法 1：固定窗口 + 频率数组

维护大小为 p.length() 的窗口，用 26 长度频率数组记录窗口内字符频率，与 p 的频率比较。

- ⏱ 时间复杂度：`O(n·26) = O(n)`
- 💾 空间复杂度：`O(1)`

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        List<Integer> result = new ArrayList<>();
        if (s.length() < p.length()) return result;
        int[] pCount = new int[26], sCount = new int[26];
        for (char c : p.toCharArray()) pCount[c - 'a']++;
        int window = p.length();
        for (int i = 0; i < s.length(); i++) {
            sCount[s.charAt(i) - 'a']++;
            if (i >= window) sCount[s.charAt(i - window) - 'a']--;
            if (i >= window - 1 && Arrays.equals(sCount, pCount))
                result.add(i - window + 1);
        }
        return result;
    }
}
```

**关键点：**
- 窗口大小固定 = p.length()
- 用 Arrays.equals 比较频率数组
- i >= window 时移除窗口最左字符

**执行步骤：**
1. 统计 p 的字符频率
1. 滑动窗口遍历 s
1. 窗口满时比较频率数组
1. 匹配则记录起始位置


## ⚠️ 易错点

- 窗口大小固定为 `len(p)`，不是变长窗口，不能用"扩展-收缩"模式。
- 比较频率时，Python 用 `Counter` 直接 `==` 比较（会忽略值为 0 的 key）；Java 需要逐位比较。
- 窗口左边界计算是 `i - len(p) + 1`，不是 `i - len(p)`。
- `s` 比 `p` 短时要直接返回空列表。

- s 长度小于 p 时直接返回空

- 频率数组比较用 Arrays.equals

## 📊 复杂度分析

- 时间复杂度：`O(n)`。窗口滑动 `O(n)`，每次比较频率 `O(26)` = `O(1)`。
- 空间复杂度：`O(1)`。频率数组固定 26 个位置。
