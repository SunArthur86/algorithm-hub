# 19. 删除链表的倒数第 N 个结点

> 难度：**Medium** | 分类：链表 | 标签：链表, 双指针

> 🔗 力扣链接：[https://leetcode.cn/problems/remove-nth-node-from-end-of-list/](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/)

## 📝 题目描述

给你一个链表，删除倒数第 `n` 个结点，并返回链表头。

比如链表 `1→2→3→4→5`，删除倒数第 2 个（值为 4），结果是 `1→2→3→5`。

- 题号：19
- 题目名：删除链表的倒数第 N 个结点
- 难度：中等
- 题型：链表
- 为什么选这题：它训练"快慢指针拉开距离"的技巧，是链表操作的经典模式。

## 🧠 思考路径

朴素做法是先遍历一遍算出链表长度 `L`，然后再遍历到第 `L-n` 个结点删掉。需要两遍遍历。

用快慢指针可以一遍搞定：快指针先走 `n` 步，然后快慢指针同时走。当快指针到达链表末尾时，慢指针刚好停在倒数第 `n+1` 个位置（待删除节点的前一个），直接跳过待删除节点即可。

关键细节：用哨兵节点（dummy head）指向头结点，这样删除头结点的边界情况和删除中间节点用同一套逻辑处理。

## 💻 Java 题解

### 解法 1：快慢指针 + dummy

快指针先走 n 步，然后快慢同时走。快指针到 null 时，慢指针的 next 就是要删除的。用 dummy 处理删头节点的情况。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(1)`

```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode dummy = new ListNode(0, head);
        ListNode fast = dummy, slow = dummy;
        for (int i = 0; i < n; i++) fast = fast.next;
        while (fast.next != null) { fast = fast.next; slow = slow.next; }
        slow.next = slow.next.next;
        return dummy.next;
    }
}
```

**关键点：**
- dummy 处理删除头节点
- fast 先走 n 步
- fast.next!=null 条件让 slow 停在被删节点前一个

**执行步骤：**
1. dummy 指向 head
1. fast 先走 n 步
1. fast 和 slow 同时走直到 fast.next=null
1. 删除 slow.next


## ⚠️ 易错点

- 必须用哨兵节点。否则删除头结点时（即倒数第 `L` 个），`slow` 没有前驱节点可以操作。
- 快指针先走 `n` 步，不是 `n-1` 步。因为我们要让 `slow` 停在待删除节点的前一个位置。
- `n` 保证是合法的（1 ≤ n ≤ 链表长度），不需要额外检查。

- 不用 dummy 时删除头节点会出错

- fast 先走 n 步不是 n+1 步

## 📊 复杂度分析

- 时间复杂度：`O(L)`。`L` 是链表长度，一遍遍历。
- 空间复杂度：`O(1)`。
