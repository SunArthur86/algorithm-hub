# 114. 二叉树展开为链表

> 难度：**Medium** | 分类：二叉树 | 标签：栈, 树, 深度优先搜索, 链表, 二叉树

> 🔗 力扣链接：[https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/)

## 📝 题目描述

给你一棵二叉树，请你将它展开为一条单链表。展开的规则是：按照前序遍历的顺序，所有节点只有右子节点，左子节点全部为空。要求原地修改，不返回新树。

直观理解就是：把树"拍扁"，让它变成一条向右延伸的链。

- 题号：114
- 题目名：二叉树展开为链表
- 难度：中等
- 题型：二叉树
- 为什么选这题：它考查的是对二叉树指针操作的理解，要求原地修改，是面试中高频的"手速题"。

## 🧠 思考路径

前序遍历的顺序是"根 → 左 → 右"。展开后，每个节点的右指针指向前序遍历中的下一个节点。

**思路一：先序遍历 + 数组记录**。先用前序遍历把所有节点按顺序存进数组，然后遍历数组，把每个节点的左指针置空、右指针指向下一个节点。这需要 `O(n)` 额外空间。

**思路二：递归后序遍历（推荐）**。从后往前处理。利用一个全局变量 `prev` 记录上一个处理过的节点。按"右 → 左 → 根"的顺序遍历（前序遍历的逆序），每处理一个节点就把它接到 `prev` 前面。这样不需要额外数组。

**思路三：迭代法（Morris 风格）**。找到当前节点左子树的最右节点，把当前节点的右子树接到这个最右节点的右边，然后把左子树移到右边。`O(1)` 空间。

## 💻 Java 题解

### 解法 1：反向后序遍历（O(1)空间）

按 右→左→根 的顺序遍历（后序的逆序），用全局 prev 指针连接。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(1)`

```java
class Solution {
    TreeNode prev = null;
    public void flatten(TreeNode root) {
        if (root == null) return;
        flatten(root.right);
        flatten(root.left);
        root.right = prev;
        root.left = null;
        prev = root;
    }
}
```

**关键点：**
- 先右后左的后序遍历
- prev 指向前一个处理的节点
- 处理顺序：最右→→根

**执行步骤：**
1. 递归右子树
1. 递归左子树
1. root.right=prev, root.left=null
1. prev=root


## ⚠️ 易错点

- 必须先把右子树接好，再移动左子树。顺序反了会丢失右子树。
- 迭代法中，找"左子树的最右节点"时，是沿着右指针走到底，不是沿左指针。
- 递归法必须按"右 → 左 → 根"的顺序处理，不能按正常的"根 → 左 → 右"。因为我们需要从尾部向前拼接。
- 题目要求原地修改，不要新建节点。

- 先递归右再递归左

- 记得 left=null

## 📊 复杂度分析

- 时间复杂度：`O(n)`。每个节点只被访问一次。
- 空间复杂度：递归法 `O(h)`（递归栈），迭代法 `O(1)`。
