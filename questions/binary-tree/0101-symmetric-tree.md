# 101. 对称二叉树

> 难度：**Easy** | 分类：二叉树 | 标签：树, 深度优先搜索, 广度优先搜索, 二叉树

> 🔗 力扣链接：[https://leetcode.cn/problems/symmetric-tree/](https://leetcode.cn/problems/symmetric-tree/)

## 📝 题目描述

给你一棵二叉树的根节点 `root`，判断它是否是轴对称的。

对称的定义：如果把这棵树沿中轴线对折，左右两边能完全重合。即左子树和右子树互为镜像。

- 题号：101
- 题目名：对称二叉树
- 难度：简单
- 题型：二叉树
- 为什么选这题：它把"相同的树"的思路稍作变形——从"两棵树同步遍历"变成"一棵树的左右子树镜像比较"，是递归思维的经典变体。

## 🧠 思考路径

和"相同的树"类似，只不过比较的不是"同一位置"，而是"镜像位置"。

两棵子树互为镜像，等价于：
1. 根节点值相同
2. 左子树的左子树和右子树的右子树互为镜像
3. 左子树的右子树和右子树的左子树互为镜像

即 `isMirror(left.left, right.right) and isMirror(left.right, right.left)`。

递归和迭代都可以。迭代版本用队列实现，每次弹出两个节点，依次比较它们的外侧孩子和内侧孩子。

## 💻 Java 题解

### 解法 1：递归比较镜像

比较左子树和右子树是否镜像对称。mirror(left,right): left.val==right.val 且 mirror(left.left,right.right) 且 mirror(left.right,right.left)。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(h)`

```java
class Solution {
    public boolean isSymmetric(TreeNode root) {
        return mirror(root.left, root.right);
    }
    private boolean mirror(TreeNode a, TreeNode b) {
        if (a == null && b == null) return true;
        if (a == null || b == null) return false;
        return a.val == b.val
            && mirror(a.left, b.right)
            && mirror(a.right, b.left);
    }
}
```

**关键点：**
- mirror(left.left, right.right)
- mirror(left.right, right.left)
- 同时为 null 返回 true

**执行步骤：**
1. 比较根的左右子树
1. 递归交叉比较
1. 值相等且子树镜像对称


## ⚠️ 易错点

- 镜像比较的配对是"左的左 vs 右的右"和"左的右 vs 右的左"，不是"左的左 vs 右的左"。
- 根节点本身不需要和谁比较，直接从 `root.left` 和 `root.right` 开始。
- 空树（`root == null`）是对称的，但 LeetCode 保证至少有一个节点。
- 迭代写法中队列每次弹出两个元素，入队也要成对入。

- 一个为 null 一个不为 null 返回 false

## 📊 复杂度分析

- 时间复杂度：`O(n)`。每个节点访问一次。
- 空间复杂度：`O(h)`（递归）或 `O(n)`（迭代队列）。最坏情况（完美二叉树）队列最多存 `n/2` 个节点。
