# 102. 二叉树的层序遍历

> 难度：**Medium** | 分类：二叉树 | 标签：树, 广度优先搜索, 二叉树

> 🔗 力扣链接：[https://leetcode.cn/problems/binary-tree-level-order-traversal/](https://leetcode.cn/problems/binary-tree-level-order-traversal/)

## 📝 题目描述

给你一棵二叉树的根节点 `root`，返回其节点值的层序遍历结果——即逐层从左到右访问所有节点，每一层的节点值组成一个列表。

比如：
```
    3
   / \
  9  20
    /  \
   15   7
```
返回 `[[3], [9, 20], [15, 7]]`。

- 题号：102
- 题目名：二叉树的层序遍历
- 难度：中等
- 题型：二叉树
- 为什么选这题：层序遍历（BFS）是树和图中最基础的广度优先搜索模式，几乎所有"按层处理"的题目都以此为基础。

## 🧠 思考路径

层序遍历的核心是 BFS（广度优先搜索），用队列来实现。

关键点：如何区分不同层？有两种方式：

1. **记录每层的节点数**：进入每层前，先获取当前队列长度 `size`，然后只弹出 `size` 个节点。这 `size` 个节点就是这一层的所有节点。
2. **用 `null` 做分隔符**：每层之间插入一个 `null` 标记。

方式 1 更常见也更简洁。每次处理完一层的所有节点后，这一层节点的所有孩子恰好都被加入了队列，正好是下一层的全部节点。

## 💻 Java 题解

### 解法 1：BFS 队列

用队列逐层处理。每轮记录当前层大小，全部弹出后进入下一层。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(n)`

```java
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;
        Queue<TreeNode> q = new LinkedList<>();
        q.offer(root);
        while (!q.isEmpty()) {
            int size = q.size();
            List<Integer> level = new ArrayList<>();
            for (int i = 0; i < size; i++) {
                TreeNode node = q.poll();
                level.add(node.val);
                if (node.left != null) q.offer(node.left);
                if (node.right != null) q.offer(node.right);
            }
            result.add(level);
        }
        return result;
    }
}
```

**关键点：**
- 每层先记录 size 再处理
- size 固定确保每层分离

**执行步骤：**
1. 根节点入队
1. 记录当前层 size
1. 弹出 size 个节点加入 level
1. 子节点入队


## ⚠️ 易错点

- `size = len(queue)` 必须在内层循环之前获取。如果在内层循环中用 `len(queue)` 判断，队列长度会动态变化，导致层和层混在一起。
- 根节点为空时直接返回空列表，不要入队。
- Python 中用 `deque` 的 `popleft()` 而不是 `list.pop(0)`，后者是 `O(n)` 的。
- 每一层的节点列表是新建的，不要复用同一个列表对象。

- 不记录 size 会导致层混在一起

## 📊 复杂度分析

- 时间复杂度：`O(n)`。每个节点入队出队各一次。
- 空间复杂度：`O(n)`。队列最多存储一层节点，完美二叉树最后一层有 `n/2` 个节点。
