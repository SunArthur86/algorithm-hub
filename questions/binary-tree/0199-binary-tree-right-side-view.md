# 199. 二叉树的右视图

> 难度：**Medium** | 分类：二叉树 | 标签：树, 深度优先搜索, 广度优先搜索, 二叉树

> 🔗 力扣链接：[https://leetcode.cn/problems/binary-tree-right-side-view/](https://leetcode.cn/problems/binary-tree-right-side-view/)

## 📝 题目描述

给你一棵二叉树的根节点，想象你站在树的右侧，按从上到下的顺序返回你能看到的节点值。也就是说，返回每层最右边的节点。

- 题号：199
- 题目名：二叉树的右视图
- 难度：中等
- 题型：二叉树
- 为什么选这题：它用 BFS 层序遍历取每层最后一个节点，是"层序遍历 + 变形应用"的典型代表。

## 🧠 思考路径

方法一（BFS）：层序遍历，每层从左到右依次入队。每层的最后一个出队元素就是右视图看到的节点。

方法二（DFS）：前序遍历的变体——先右子树后左子树。每层第一次被访问的节点就是最右边的。用一个变量 `depth` 记录当前深度，如果这个深度还没被记录过，就把当前节点加入结果。

BFS 更直观，DFS 更省空间（递归栈 vs 队列）。这里用 BFS 解。

## 💻 Java 题解

### 解法 1：BFS 每层最后节点

层序遍历，每层最后一个节点就是右视图看到的节点。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(n)`

```java
class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        if (root == null) return result;
        Queue<TreeNode> q = new LinkedList<>();
        q.offer(root);
        while (!q.isEmpty()) {
            int size = q.size();
            for (int i = 0; i < size; i++) {
                TreeNode node = q.poll();
                if (i == size - 1) result.add(node.val); // 每层最后一个
                if (node.left != null) q.offer(node.left);
                if (node.right != null) q.offer(node.right);
            }
        }
        return result;
    }
}
```

**关键点：**
- BFS每层最后一个节点
- i == size-1 时记录

**执行步骤：**
1. BFS层序遍历
1. 每层最后一个节点加入结果


## ⚠️ 易错点

- 每层开始前必须"快照"队列长度（`level_size = len(queue)`），因为队列在循环中会变化。
- 不是"只遍历右子树"——有些节点可能没有右子节点，右视图的节点在左子树里。
- 空树要返回空列表，不是 `null`。

- 空树返回空列表

## 📊 复杂度分析

- 时间复杂度：`O(n)`，每个节点访问一次。
- 空间复杂度：`O(w)`，`w` 是树的最大宽度（队列大小）。
