# 108. 将有序数组转换为二叉搜索树

> 难度：**Easy** | 分类：二叉树 | 标签：树, 二叉搜索树, 数组, 分治, 二叉树

> 🔗 力扣链接：[https://leetcode.cn/problems/convert-sorted-array-to-binary-search-tree/](https://leetcode.cn/problems/convert-sorted-array-to-binary-search-tree/)

## 📝 题目描述

给你一个整数数组 `nums`，已按升序排列，请将其转换为一棵高度平衡的二叉搜索树。

"高度平衡"指的是：每个节点的左右两个子树的高度差不超过 1。

- 题号：108
- 题目名：将有序数组转换为二叉搜索树
- 难度：简单
- 题型：二叉树
- 为什么选这题：它是"有序序列 → 平衡 BST"的基础题，"取中间元素当根"的分治策略简洁优美，也是后续红黑树、AVL 树等平衡操作的起点。

## 🧠 思考路径

数组已经是有序的，而 BST 的中序遍历正好是有序序列。关键是怎么保证平衡。

要平衡，就要让左右子树的节点数尽量接近。最自然的做法是取数组的中间元素作为根，这样左半部分和右半部分的元素数量最多差 1。

递归分治：
1. 取 `nums[mid]` 作为根节点。
2. `nums[left..mid-1]` 递归构造左子树。
3. `nums[mid+1..right]` 递归构造右子树。
4. `left > right` 时返回空节点。

中间元素的选择：`mid = (left + right) // 2`（偶数时偏左）或 `mid = (left + right + 1) // 2`（偏右）。两种都能构造出合法的平衡 BST，只是形状可能不同。题目说"答案不唯一"，任选一种即可。

## 💻 Java 题解

### 解法 1：分治（取中点为根）

每次取数组中间元素作为当前根，左半部分递归构建左子树，右半部分构建右子树。这样保证平衡。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(log n)`

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        return build(nums, 0, nums.length - 1);
    }
    private TreeNode build(int[] nums, int left, int right) {
        if (left > right) return null;
        int mid = left + (right - left) / 2;
        TreeNode root = new TreeNode(nums[mid]);
        root.left = build(nums, left, mid - 1);
        root.right = build(nums, mid + 1, right);
        return root;
    }
}
```

**关键点：**
- 中点为根保证平衡
- 左半→左子树，右半→右子树
- mid = left + (right-left)/2 防溢出

**执行步骤：**
1. 取中点为根
1. 递归构建左子树
1. 递归构建右子树


## ⚠️ 易错点

- `mid = left + (right - left) / 2` 可以防止整数溢出（虽然 Python 不存在这个问题，但 Java 中是良好习惯）。
- 左子树的范围是 `[left, mid-1]`，右子树是 `[mid+1, right]`，`mid` 本身已经用作根了，不要重复包含。
- 答案不唯一：选偏左的中间还是偏右的中间都行，LeetCode 接受任何合法的平衡 BST。
- 不要忘记 `left > right` 的终止条件，否则会无限递归。

- mid = left + (right-left)/2 防止整数溢出

## 📊 复杂度分析

- 时间复杂度：`O(n)`。每个元素恰好被访问一次作为某个节点的值。
- 空间复杂度：`O(log n)`。递归栈深度等于树的高度 `O(log n)`（因为树是平衡的）。
