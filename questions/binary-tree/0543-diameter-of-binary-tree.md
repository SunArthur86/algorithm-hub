# 543. 二叉树的直径

> 难度：**Easy** | 分类：二叉树 | 标签：树, 深度优先搜索, 二叉树

> 🔗 力扣链接：[https://leetcode.cn/problems/diameter-of-binary-tree/](https://leetcode.cn/problems/diameter-of-binary-tree/)

## 📝 题目描述

给你一棵二叉树的根节点 `root`。返回这棵树的直径长度——即任意两个节点之间的最长路径的边数。这条路径可能经过也可能不经过根节点。

- 题号：543
- 题目名：二叉树的直径
- 难度：简单
- 题型：二叉树
- 为什么选这题：它是"树形 DP"的入门题，核心思想是"每个节点返回向下的最大深度，同时用左右深度之和更新全局答案"。

## 🧠 思考路径

直径就是树中最长的路径。对于每个节点，经过它的最长路径 = 左子树最大深度 + 右子树最大深度。

所以做法是：对每个节点，计算它的左子树深度和右子树深度，两者之和就是经过该节点的路径长度。用全局变量维护所有节点中的最大值。

每个节点返回以它为根的子树的最大深度（`max(左深度, 右深度) + 1`），供父节点使用。

## 💻 Java 题解

### 解法 1：DFS + 全局最大值

DFS 返回以当前节点为端点的最大深度。过程中用 left+right 更新直径最大值。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(h)`

```java
class Solution {
    int maxDiameter = 0;
    public int diameterOfBinaryTree(TreeNode root) {
        depth(root);
        return maxDiameter;
    }
    private int depth(TreeNode node) {
        if (node == null) return 0;
        int left = depth(node.left);
        int right = depth(node.right);
        maxDiameter = Math.max(maxDiameter, left + right);
        return 1 + Math.max(left, right);
    }
}
```

**关键点：**
- 直径 = 经过某节点的 left+right
- depth返回子树深度
- 全局变量更新最大直径

**执行步骤：**
1. DFS求深度
1. 过程中 left+right 更新直径
1. 返回 1+max(left,right)


## ⚠️ 易错点

- 直径的定义是"边数"不是"节点数"。`left + right` 已经是边数了，因为深度是从当前节点开始往下数的边数。
- 直径不一定经过根节点，所以必须检查每个节点，不能只检查根。
- 空节点的深度返回 0，不是 -1。

- 直径是边数不是节点数

- 不一定经过根节点

## 📊 复杂度分析

- 时间复杂度：`O(n)`，每个节点访问一次。
- 空间复杂度：`O(h)`，递归栈深度。
