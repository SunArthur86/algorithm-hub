# 118. 杨辉三角

> 难度：**Easy** | 分类：动态规划 | 标签：数组, 动态规划

> 🔗 力扣链接：[https://leetcode.cn/problems/pascals-triangle/](https://leetcode.cn/problems/pascals-triangle/)

## 📝 题目描述

给定一个非负整数 `numRows`，生成杨辉三角的前 `numRows` 行。

杨辉三角的规则：第一行是 `[1]`，之后每一行的第一个和最后一个数都是 1，中间的每个数等于它"上方"和"左上方"两个数之和。

- 题号：118
- 题目名：杨辉三角
- 难度：简单
- 题型：动态规划
- 为什么选这题：它是二维 DP 最直观的入门题，每个位置的值完全由上一行决定，状态转移关系一目了然。

## 🧠 思考路径

这题几乎不需要复杂的思考——每一行的内容完全由上一行决定。`row[i] = prev_row[i-1] + prev_row[i]`，边界位置填 1。

动态规划的特征很明显：有明确的递推关系（状态转移），有边界条件（首尾为 1），有计算顺序（从上到下逐行生成）。这也是为什么它常被用作 DP 入门题。

实现时可以直接逐行构建：从第一行 `[1]` 开始，每生成新的一行，就把它加入结果列表。

## 💻 Java 题解

### 解法 1：逐行构建

每行首尾为1，中间元素=上一行[j-1]+上一行[j]。

- ⏱ 时间复杂度：`O(n²)`
- 💾 空间复杂度：`O(n²)`

```java
class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> result = new ArrayList<>();
        for (int i = 0; i < numRows; i++) {
            List<Integer> row = new ArrayList<>();
            for (int j = 0; j <= i; j++) {
                if (j == 0 || j == i) row.add(1);
                else row.add(result.get(i-1).get(j-1) + result.get(i-1).get(j));
            }
            result.add(row);
        }
        return result;
    }
}
```

**关键点：**
- 首尾为1
- 中间=上一行相邻两数之和

**执行步骤：**
1. 逐行构建
1. 首尾填1
1. 中间取上一行之和


## ⚠️ 易错点

- 第 `i` 行有 `i + 1` 个元素（从 0 开始计数），不是 `i` 个。
- 中间元素的索引范围是 `[1, i-1]`（`i` 从 0 开始），不要越界。
- 初始化每行时可以先填满 1，再修改中间位置，代码更简洁。

- 注意索引边界

## 📊 复杂度分析

- 时间复杂度：`O(numRows²)`。第 `i` 行需要 `O(i)` 时间，总和是 `1 + 2 + ... + numRows = O(numRows²)`。
- 空间复杂度：`O(numRows²)`。存储整个三角需要同等大小的空间（不算返回值则为 `O(numRows)` 用于当前行）。

取经路上每往前走一步（每一行），可能遇到的磨难组合就更多。第一层只有一种磨难，第二层有两种组合，第三层每种磨难又能和前面的组合出新的分支。每一层的磨难数等于上一层"相邻两次经历的磨难数之和"。观音菩萨看着这份磨难表，就知道第 `n` 天唐僧可能遇到的磨难组合有多少种。

- "磨难表"对应杨辉三角

- "相邻经历的磨难数之和"对应 `prev[i-1] + prev[i]`
- "组合数"对应组合数 `C(n, k)`
