# 279. 完全平方数

> 难度：**Medium** | 分类：动态规划 | 标签：广度优先搜索, 数学, 动态规划

> 🔗 力扣链接：[https://leetcode.cn/problems/perfect-squares/](https://leetcode.cn/problems/perfect-squares/)

## 📝 题目描述

给你一个正整数 `n`，返回和为 `n` 的最少完全平方数个数。完全平方数是 1, 4, 9, 16, ...

比如 `n = 12`，`12 = 4 + 4 + 4`，所以返回 3。

重点不在"会不会加法"，而在"怎么找到最少的分解方式"。

- 题号：279
- 题目名：完全平方数
- 难度：中等
- 题型：动态规划
- 为什么选这题：它是"完全背包"的经典变形，用"最少硬币凑金额"的思路来理解最优。

## 🧠 思考路径

这题可以看成完全背包问题：物品是完全平方数（1, 4, 9, ...），背包容量是 `n`，每个物品可以用无限次，求最少物品数。

设 `dp[i]` 表示和为 `i` 的最少完全平方数个数。转移方程：`dp[i] = min(dp[i - j*j] + 1)`，其中 `j*j <= i`。

初始条件：`dp[0] = 0`，其余设为无穷大。从 `i = 1` 开始，依次尝试所有小于等于 `i` 的完全平方数，取最小值。

还有一个数学定理（四平方和定理）：任何正整数最多可以表示为 4 个完全平方数的和。所以答案最多是 4。

所以这题的关键词不是"怎么凑"，而是"dp[i] = 所有可能的 j² 组合中取最小"。

## 💻 Java 题解

### 解法 1：动态规划

dp[i]表示i最少需要几个完全平方数。枚举所有j²<=i，dp[i]=min(dp[i-j²]+1)。

- ⏱ 时间复杂度：`O(n√n)`
- 💾 空间复杂度：`O(n)`

```java
class Solution {
    public int numSquares(int n) {
        int[] dp = new int[n + 1];
        Arrays.fill(dp, Integer.MAX_VALUE);
        dp[0] = 0;
        for (int i = 1; i <= n; i++)
            for (int j = 1; j * j <= i; j++)
                dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
        return dp[n];
    }
}
```

**关键点：**
- 枚举j*j<=i的所有j
- dp[i]=min(dp[i-j²]+1)
- 初始化MAX_VALUE

**执行步骤：**
1. 初始化dp数组
1. 双重循环枚举平方数
1. 取最小值


## ⚠️ 易错点

- `j * j` 可能溢出（当 n 很大时），Java 中要注意。
- `dp` 数组的大小是 `n + 1`，不是 `n`。
- 初始值不能设为 `n + 1`（如果 n 恰好是最坏情况），用 `Integer.MAX_VALUE` 更安全。

- dp[0]=0 是base case

- Arrays.fill初始化为MAX_VALUE

## 📊 复杂度分析

- 时间复杂度：`O(n * √n)`，外层 n 次，内层最多 `√n` 次。
- 空间复杂度：`O(n)`。
