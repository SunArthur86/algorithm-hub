# 84. 柱状图中最大的矩形

> 难度：**Hard** | 分类：栈 | 标签：栈, 数组, 单调栈

> 🔗 力扣链接：[https://leetcode.cn/problems/largest-rectangle-in-histogram/](https://leetcode.cn/problems/largest-rectangle-in-histogram/)

## 📝 题目描述

给你一个非负整数数组 `heights`，表示柱状图中各柱子的高度，每根柱子宽度为 `1`。求柱状图中能框出的最大矩形面积。

比如 `heights = [2,1,5,6,2,3]`，最大矩形面积是 `10`（高度 5 和 6 的两根柱子框出 `5×2=10`）。

- 题号：84
- 题目名：柱状图中最大的矩形
- 难度：困难
- 题型：单调栈
- 为什么选这题：它是单调栈的巅峰之作，理解了"以每根柱子为高度的最大矩形"这个核心思想，单调栈就算真正入门了。

## 🧠 思考路径

暴力方法是枚举每对左右边界，计算矩形面积，`O(n²)`。

换个角度：以第 `i` 根柱子的高度为矩形高度，矩形能向左右扩展多远？答案是：向左到第一根比它矮的柱子右边一位，向右到第一根比它矮的柱子左边一位。即找到每个柱子的"左右边界"。

"找左边/右边第一个比它小的元素"——这正是单调栈的拿手好戏。维护一个单调递增栈，当新来的柱子比栈顶矮时，栈顶柱子的右边界就确定了（就是当前柱子）。弹出栈顶后，新的栈顶就是它的左边界。面积 = `heights[栈顶] × (right - left - 1)`。

## 💻 Java 题解

### 解法 1：单调递增栈

栈存索引，高度递增。遇到比栈顶矮的柱子时弹出，计算以弹出柱子为高的矩形面积。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(n)`

```java
class Solution {
    public int largestRectangleArea(int[] heights) {
        Deque<Integer> stack = new ArrayDeque<>();
        int maxArea = 0;
        for (int i = 0; i <= heights.length; i++) {
            int h = i == heights.length ? 0 : heights[i];
            while (!stack.isEmpty() && h < heights[stack.peek()]) {
                int height = heights[stack.pop()];
                int width = stack.isEmpty() ? i : i - stack.peek() - 1;
                maxArea = Math.max(maxArea, height * width);
            }
            stack.push(i);
        }
        return maxArea;
    }
}
```

**关键点：**
- 单调递增栈
- i==n 时 h=0 强制清空栈
- 宽度 = i - stack.peek() - 1

**执行步骤：**
1. 遍历到i+1（含末尾哨兵0）
1. 比栈顶矮→弹出计算面积
1. 宽=当前i-新栈顶-1


## ⚠️ 易错点

- 哨兵 `0` 不能忘。没有它，最后栈里可能还有柱子没被处理。
- 宽度计算：弹出后如果栈为空，宽度就是 `i`（从 0 到 i-1）；否则是 `i - stack[-1] - 1`。
- 栈里存的是下标不是高度，取出后要用 `heights[stack[-1]]` 来获取高度。

- 末尾哨兵h=0确保栈清空

- 栈空时宽度=i

## 📊 复杂度分析

- 时间复杂度：`O(n)`，每个元素最多入栈出栈各一次。
- 空间复杂度：`O(n)`，栈的大小。
