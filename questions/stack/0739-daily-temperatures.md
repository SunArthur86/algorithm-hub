# 739. 每日温度

> 难度：**Medium** | 分类：栈 | 标签：栈, 数组, 单调栈

> 🔗 力扣链接：[https://leetcode.cn/problems/daily-temperatures/](https://leetcode.cn/problems/daily-temperatures/)

## 📝 题目描述

给你整数数组 `temperatures`，`temperatures[i]` 表示第 i 天气温。返回数组 `answer`，`answer[i]` 表示要等多少天才遇到更高温度。之后不会升温则填 0。

- 题号：739
- 题目名：每日温度
- 难度：中等
- 题型：单调栈
- 为什么选这题：这是"下一个更大元素"模型的代表题，单调栈模板价值非常高，刷懂这一题能带出一大片题。

## 🧠 思考路径

本质是"找右边第一个比它大的元素"。用单调栈存下标，保持栈中温度单调递减。新温度进来时，只要比栈顶高就弹出栈顶并计算间隔天数。每元素最多进出栈一次，`O(n)`。

## 💻 Java 题解

### 解法 1：单调递减栈

栈存索引，栈内温度递减。遇到比栈顶高的温度时弹出，计算天数差。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(n)`

```java
class Solution {
    public int[] dailyTemperatures(int[] temperatures) {
        int n = temperatures.length;
        int[] result = new int[n];
        Deque<Integer> stack = new ArrayDeque<>();
        for (int i = 0; i < n; i++) {
            while (!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]) {
                int idx = stack.pop();
                result[idx] = i - idx;
            }
            stack.push(i);
        }
        return result;
    }
}
```

**关键点：**
- 单调递减栈
- 遇到更高温弹出
- 天数差 = 当前索引 - 弹出索引

**执行步骤：**
1. 遍历每天温度
1. 比栈顶高时弹出计算天数差
1. 当前索引入栈


## ⚠️ 易错点

1. **栈里存下标不是温度值**
   因为要算间隔天数 `i - idx`。

2. **维护的是递减栈**
   栈顶是最近还没找到更高温度的那天。

3. **比较条件是严格小于**
   相等不算"更高"，用 `<` 不是 `<=`。

4. **栈里剩余元素答案保持 0**
   它们右边已没有更高温度。

- 栈存索引不是温度值

- 用while不是if
