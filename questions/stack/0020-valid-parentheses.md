# 20. 有效的括号

> 难度：**Easy** | 分类：栈 | 标签：栈, 字符串

> 🔗 力扣链接：[https://leetcode.cn/problems/valid-parentheses/](https://leetcode.cn/problems/valid-parentheses/)

## 📝 题目描述

给你一个只包含 `(`, `)`, `{`, `}`, `[`, `]` 的字符串 `s`，判断括号是否有效：左括号必须用相同类型的右括号闭合，且必须按正确顺序闭合。

- 题号：20
- 题目名：有效的括号
- 难度：简单
- 题型：栈
- 为什么选这题：它是最经典的栈应用题，训练"后进先出匹配"的直觉。

## 🧠 思考路径

括号匹配的核心规律是：最后出现的左括号，最先被闭合。这正好是栈的"后进先出"特性。

遇到左括号就入栈，遇到右括号就检查栈顶是否是对应的左括号：是就弹出，不是就不合法。最后栈为空说明所有括号都配对了。

## 💻 Java 题解

### 解法 1：栈

遍历字符串，左括号入栈，右括号弹出栈顶检查是否匹配。最后栈必须为空。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(n)`

```java
class Solution {
    public boolean isValid(String s) {
        Deque<Character> stack = new ArrayDeque<>();
        Map<Character, Character> pairs = Map.of(')', '(', ']', '[', '}', '{');
        for (char c : s.toCharArray()) {
            if (pairs.containsValue(c)) stack.push(c);
            else if (pairs.containsKey(c)) {
                if (stack.isEmpty() || stack.pop() != pairs.get(c)) return false;
            }
        }
        return stack.isEmpty();
    }
}
```

**关键点：**
- 右括号→左括号的映射
- 弹出时检查匹配
- 最后栈为空才有效

**执行步骤：**
1. 左括号入栈
1. 右括号弹出栈顶比较
1. 遍历完栈必须为空


## ⚠️ 易错点

- 遇到右括号时栈可能为空（如 `"]"`），要先检查栈是否为空再取栈顶。
- 遍历结束后必须检查栈是否为空。如果有剩余的左括号（如 `"(["`），说明没有配对完。
- 奇数长度的字符串一定不合法，可以提前返回。

- 右括号时栈可能为空

- Map.of 是Java9+特性

## 📊 复杂度分析

- 时间复杂度：`O(n)`。遍历字符串一次。
- 空间复杂度：`O(n)`。栈最多存 `n/2` 个左括号。

对应关系是：

- "打开的盒子"对应左括号（入栈）

- "最里层先关"对应栈的 LIFO 特性
- "有盒子没关"对应栈非空

记忆的关键是：后开先关，栈来匹配。
