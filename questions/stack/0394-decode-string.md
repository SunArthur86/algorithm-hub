# 394. 字符串解码

> 难度：**Medium** | 分类：栈 | 标签：栈, 递归, 字符串

> 🔗 力扣链接：[https://leetcode.cn/problems/decode-string/](https://leetcode.cn/problems/decode-string/)

## 📝 题目描述

给你一个编码后的字符串，规则是 `k[encoded_string]` 表示把 `encoded_string` 重复 `k` 次。编码可以嵌套，比如 `3[a2[c]]` 解码后是 `accaccacc`。

你需要把整个字符串完全解码展开，返回最终结果。

- 题号：394
- 题目名：字符串解码
- 难度：Medium
- 题型：栈
- 为什么选这题：它是栈处理嵌套结构的经典题，从括号匹配升级到"带重复次数的嵌套展开"，能深度训练栈的运用能力。

## 🧠 思考路径

最直接的想法：遇到嵌套就要"从内向外"展开。先找到最内层的括号对，展开它，再处理外层。但这样需要反复扫描字符串，嵌套层数深时效率很低。

用栈来优化：从左到右遍历字符串，遇到字母就追加到当前结果，遇到数字就解析完整数字，遇到 `[` 就把当前结果和数字压栈、重新开始记录新的一层，遇到 `]` 就弹栈——把弹出的一层结果重复 `k` 次后拼到上一层的后面。

关键洞察：遇到 `[` 时保存现场（当前已积累的字符串和重复次数），开始记录新一层的内容；遇到 `]` 时恢复现场并把新一层的结果展开后拼回去。栈天然地处理了这种"保存-恢复"的嵌套关系。

用两个栈（或一个栈存元组）分别记录每一层的字符串前缀和重复次数。当前层的字符串遇到 `]` 时完成展开，拼到上一层字符串的末尾。

## 💻 Java 题解

### 解法 1：双栈

遍历字符串：数字入数字栈，字母拼接当前串，[时把当前串和数字入栈并重置，]时弹出拼接。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(n)`

```java
class Solution {
    public String decodeString(String s) {
        Deque<Integer> countStack = new ArrayDeque<>();
        Deque<StringBuilder> strStack = new ArrayDeque<>();
        StringBuilder curr = new StringBuilder();
        int num = 0;
        for (char c : s.toCharArray()) {
            if (Character.isDigit(c)) num = num * 10 + c - '0';
            else if (c == '[') { countStack.push(num); strStack.push(curr); curr = new StringBuilder(); num = 0; }
            else if (c == ']') {
                int repeat = countStack.pop();
                StringBuilder prev = strStack.pop();
                for (int i = 0; i < repeat; i++) prev.append(curr);
                curr = prev;
            } else curr.append(c);
        }
        return curr.toString();
    }
}
```

**关键点：**
- 两个栈：数字栈+字符串栈
- [时保存当前状态并重置
- ]时弹出重复拼接

**执行步骤：**
1. 数字累加
1. [: 入栈+重置
1. ]: 弹出重复拼接
1. 字母: 拼接当前串


## ⚠️ 易错点

- 数字可能是多位的（如 `10[abc]`），不能只取单个字符做数字，要累加解析。
- 入栈时机是遇到 `[`，不是遇到数字。数字只是准备阶段，`[` 才是"新一层开始"的信号。
- Python 中字符串拼接 `cur_str * repeat` 是直接重复，但 Java 中要用 `StringBuilder` 循环追加，避免大量中间对象。

- 数字可能多位——num=num*10+c-'0'

- ]时是prev.append(curr)不是反过来

## 📊 复杂度分析

- 时间复杂度：`O(n)`，其中 `n` 是解码后字符串的长度（最坏情况下展开操作是线性的）。
- 空间复杂度：`O(m)`，其中 `m` 是最大嵌套深度，栈的空间与嵌套层数成正比。
