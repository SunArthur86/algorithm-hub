注意这个 for 循环，就是在连续的空间线性搜索，这就是⼆分查找可以发挥
作⽤的标志。由于我们要求的是最⼩速度，所以可以⽤⼀个搜索左侧边界的
⼆分查找来代替线性搜索，提升效率：
int minEatingSpeed(int[] piles, int H) {
    // 套⽤搜索左侧边界的算法框架
    int left = 1, right = getMax(piles) + 1;
    while (left < right) {
        // 防⽌溢出
        int mid = left + (right - left) / 2;
        if (canFinish(piles, mid, H)) {
            right = mid;
        } else {
            left = mid + 1;
        }
    }
    return left;
}
