「反转以  a  为头结点的链表」其实就是「反转  a  到 null 之间的结点」，
那么如果让你「反转  a  到  b  之间的结点」，你会不会？
只要更改函数签名，并把上⾯的代码中  null  改成  b  即可：
/** 反转区间 [a, b) 的元素，注意是左闭右开 */
ListNode reverse(ListNode a, ListNode b) {
    ListNode pre, cur, nxt;
    pre = null; cur = a; nxt = a;
    // while 终⽌的条件改⼀下就⾏了
    while (cur != b) {
        nxt = cur.next;
        cur.next = pre;
        pre = cur;
        cur = nxt;
    }
    // 返回反转后的头结点
    return pre;
}
