e:T1bab,# 54. 螺旋矩阵

> 难度：**Medium** | 分类：矩阵 | 标签：数组, 矩阵, 模拟

> 🔗 力扣链接：[https://leetcode.cn/problems/spiral-matrix/](https://leetcode.cn/problems/spiral-matrix/)

## 📝 题目描述

给你一个 `m` 行 `n` 列的矩阵，按照顺时针螺旋的顺序（从左上角开始：右→下→左→上→右……），把所有元素取出来，放到一个一维数组里返回。

- 题号：54
- 题目名：螺旋矩阵
- 难度：中等
- 题型：矩阵
- 为什么选这题：矩阵按层遍历是最基础的二维坐标操作技巧，面试出场率极高。

## 🧠 思考路径

最直观的想法：模拟走的过程。从左上角出发，先一路向右走到头，再一路向下走到头，再一路向左走到头，再一路向上走——这就是"一层"。

走完一层后，外圈已经被掏空了，内圈缩小一圈，继续重复同样的操作。用四个变量 `top`、`bottom`、`left`、`right` 表示当前还没遍历的区域的边界。

每走完一条边，对应的边界就往内缩一格。当 `top > bottom` 或 `left > right` 时，说明全部走完了。

需要注意的是：非方阵（行数和列数不同）时，最后可能只剩一行或一列，要避免重复取。

## 💻 Java 题解

### 解法 1：边界收缩法

维护四个边界，按 右→下→左→上 顺序遍历，每次遍历完一条边就收缩对应边界。

- ⏱ 时间复杂度：`O(m×n)`
- 💾 空间复杂度：`O(1)`

```java
class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        List<Integer> result = new ArrayList<>();
        int top = 0, bottom = matrix.length - 1;
        int left = 0, right = matrix[0].length - 1;
        while (top <= bottom && left <= right) {
            for (int j = left; j <= right; j++) result.add(matrix[top][j]);
            top++;
            for (int i = top; i <= bottom; i++) result.add(matrix[i][right]);
            right--;
            if (top <= bottom) {
                for (int j = right; j >= left; j--) result.add(matrix[bottom][j]);
                bottom--;
            }
            if (left <= right) {
                for (int i = bottom; i >= top; i--) result.add(matrix[i][left]);
                left++;
            }
        }
        return result;
    }
}
```

**关键点：**
- 四边界收缩
- 右→下→左→上循环
- 下和左方向需要额外检查边界

**执行步骤：**
1. 初始化 top/bottom/left/right
1. 向右遍历 top 行，top++
1. 向下遍历 right 列，right--
1. 向左遍历 bottom 行（如果还有），bottom--
1. 向上遍历 left 列（如果还有），left++


## ⚠️ 易错点

- 向左和向上遍历之前，必须检查边界条件（`top <= bottom` 和 `left <= right`），否则在非方阵时会重复取元素。比如 3×4 矩阵走完一圈后可能只剩中间一行，如果不检查就会倒着再取一遍。
- 循环变量的范围要仔细：向右是 `left → right`，向左是 `right → left`，不要搞反方向或漏端点。
- 边界缩小（`top++`、`right--` 等）必须在遍历完当前边之后立刻做，不要放错位置。

- 向左和向上时需要 if 检查防止重复遍历

- 单行/单列的边界条件

## 📊 复杂度分析

- 时间复杂度：`O(m × n)`。每个元素恰好被访问一次。
- 空间复杂度：`O(1)`（不算返回结果数组），只用了四个边界变量。
