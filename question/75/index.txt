e:T192e,# 75. 颜色分类

> 难度：**Medium** | 分类：技巧 | 标签：数组, 双指针, 排序

> 🔗 力扣链接：[https://leetcode.cn/problems/sort-colors/](https://leetcode.cn/problems/sort-colors/)

## 📝 题目描述

给你一个数组，里面只有 `0`、`1`、`2` 三种值，分别代表红、白、蓝。要求原地排序，使得所有 `0` 在最前面，`1` 在中间，`2` 在最后面。不能使用内置排序函数。

- 题号：75
- 题目名：颜色分类
- 难度：中等
- 题型：双指针
- 为什么选这题：它是"荷兰国旗问题"的标准实现，用三个指针做一趟扫描完成三分区排序。

## 🧠 思考路径

最简单的方法是计数排序：数一下各有几个 `0`、`1`、`2`，然后重写数组。但这需要遍历两遍。

更好的方法是荷兰国旗算法。用三个指针把数组分成四个区域：`[0, low)` 全是 `0`，`[low, mid)` 全是 `1`，`[mid, high]` 是待处理区域，`(high, end]` 全是 `2`。`mid` 指针从左到右扫描：遇到 `0` 就和 `low` 交换然后都右移，遇到 `1` 就直接跳过，遇到 `2` 就和 `high` 交换然后 `high` 左移。

一趟扫描就排好了。

## 💻 Java 题解

### 解法 1：三指针（荷兰国旗）

三个指针：left指向0的右边界，right指向2的左边界，curr遍历。遇到0交换到left，遇到2交换到right。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(1)`

```java
class Solution {
    public void sortColors(int[] nums) {
        int left = 0, right = nums.length - 1, curr = 0;
        while (curr <= right) {
            if (nums[curr] == 0) {
                swap(nums, curr++, left++);
            } else if (nums[curr] == 2) {
                swap(nums, curr, right--);
            } else {
                curr++;
            }
        }
    }
    private void swap(int[] nums, int i, int j) {
        int t = nums[i]; nums[i] = nums[j]; nums[j] = t;
    }
}
```

**关键点：**
- 遇到0: swap(curr,left),curr++left++
- 遇到2: swap(curr,right),right--
- 遇到1: curr++

**执行步骤：**
1. curr遍历
1. 0→交换到左边
1. 2→交换到右边
1. 1→跳过


## ⚠️ 易错点

- 遇到 `2` 和 `high` 交换后，`mid` 不能右移，因为换过来的元素还不知道是什么，需要再判断一次。
- 遇到 `0` 时是和 `low` 交换，不是和最左边交换。`low` 和 `mid` 可能不同步。
- 循环条件是 `mid <= high`，不是 `mid < high`，因为 `mid == high` 时那个元素还没处理。

- 交换2后curr不自增——交换来的元素还需检查

- while条件curr<=right

## 📊 复杂度分析

- 时间复杂度：`O(n)`，每个元素最多被访问一次。
- 空间复杂度：`O(1)`，原地操作。
