e:T1b18,# 5. 最长回文子串

> 难度：**Medium** | 分类：多维动态规划 | 标签：字符串, 动态规划

> 🔗 力扣链接：[https://leetcode.cn/problems/longest-palindromic-substring/](https://leetcode.cn/problems/longest-palindromic-substring/)

## 📝 题目描述

给你一个字符串 `s`，找出其中最长的回文子串。回文就是正着读和反着读一样的字符串，比如 `"aba"`、`"abba"`。

- 题号：5
- 题目名：最长回文子串
- 难度：中等
- 题型：动态规划
- 为什么选这题：它是动态规划和中心扩展两种思路的经典交汇点，训练"从内向外在字符串上扩展"的思维。

## 🧠 思考路径

最朴素的做法是枚举所有子串，判断每个是否是回文，取最长的。但子串个数是 `O(n²)`，每个判断要 `O(n)`，总复杂度 `O(n³)`。

观察回文的性质：如果一个字符串是回文，那么去掉首尾两个字符后，剩下的也是回文。反过来，如果已知中间一段是回文，只要首尾两个字符相同，就能向外扩展一圈。这就是"中心扩展法"的思路。

另一种方式是动态规划。定义 `dp[i][j]` 表示 `s[i..j]` 是否是回文。转移方程是 `dp[i][j] = (s[i] == s[j]) and dp[i+1][j-1]`。按子串长度从小到大填表即可。

## 💻 Java 题解

### 解法 1：中心扩展法

对每个中心（奇数和偶数两种），向两边扩展找最长回文。

- ⏱ 时间复杂度：`O(n²)`
- 💾 空间复杂度：`O(1)`

```java
class Solution {
    public String longestPalindrome(String s) {
        int start = 0, maxLen = 1;
        for (int i = 0; i < s.length(); i++) {
            int len1 = expand(s, i, i);     // 奇数
            int len2 = expand(s, i, i + 1); // 偶数
            int len = Math.max(len1, len2);
            if (len > maxLen) {
                maxLen = len;
                start = i - (len - 1) / 2;
            }
        }
        return s.substring(start, start + maxLen);
    }
    private int expand(String s, int l, int r) {
        while (l >= 0 && r < s.length() && s.charAt(l) == s.charAt(r)) { l--; r++; }
        return r - l - 1;
    }
}
```

**关键点：**
- 两种中心：i和i,i+1
- expand返回长度
- start = i-(len-1)/2

**执行步骤：**
1. 遍历每个中心
1. 奇偶两种扩展
1. 更新最大值和起点


## ⚠️ 易错点

- 中心扩展必须同时考虑奇数和偶数长度两种情况。奇数以单个字符为中心，偶数以两个相邻字符的间隙为中心。
- 扩展结束后，`left` 和 `right` 都多走了一步，所以实际的回文区间是 `[left+1, right-1]`。
- 单字符字符串本身就是回文，需要单独处理或保证循环正确覆盖。

- start = i-(len-1)/2 不是 i-len/2

- expand返回r-l-1 不是r-l+1

## 📊 复杂度分析

- 时间复杂度：`O(n²)`。每个字符作为中心最多扩展 `O(n)` 次。
- 空间复杂度：`O(1)`。只用了常数个变量。
