e:T1ac0,# 2. 两数相加

> 难度：**Medium** | 分类：链表 | 标签：递归, 链表, 数学

> 🔗 力扣链接：[https://leetcode.cn/problems/add-two-numbers/](https://leetcode.cn/problems/add-two-numbers/)

## 📝 题目描述

给你两个链表，每个节点存一位数字，整体按逆序排列（个位在头）。你要把这两个"链表数"加起来，结果也用同样的逆序链表返回。

比如 `2 → 4 → 3` 代表 342，`5 → 6 → 4` 代表 465，加起来是 807，返回 `7 → 0 → 8`。

- 题号：2
- 题目名：两数相加
- 难度：中等
- 题型：链表
- 为什么选这题：它是最经典的链表模拟题，同时训练"指针操作"和"进位处理"两个基本功。

## 🧠 思考路径

最自然的想法就是模拟竖式加法。从最低位（链表头）开始，一位一位地加，有进位就记下来带到下一位。

关键在于：两个链表长度可能不一样，有一个先走完；走完的那个相当于不断补零。最后如果还剩进位，要额外挂一个节点。

整个过程只需要一个指针沿着两条链同步往后走，一个变量 `carry` 记进位。不需要先把链表转成数字再相加，因为数字可能非常大，直接用链表操作更安全。

## 💻 Java 题解

### 解法 1：模拟加法 + 进位

遍历两个链表，逐位相加加上进位。 carry = sum / 10, new digit = sum % 10。

- ⏱ 时间复杂度：`O(max(m,n))`
- 💾 空间复杂度：`O(max(m,n))`

```java
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0), curr = dummy;
        int carry = 0;
        while (l1 != null || l2 != null || carry != 0) {
            int sum = carry;
            if (l1 != null) { sum += l1.val; l1 = l1.next; }
            if (l2 != null) { sum += l2.val; l2 = l2.next; }
            curr.next = new ListNode(sum % 10);
            carry = sum / 10;
            curr = curr.next;
        }
        return dummy.next;
    }
}
```

**关键点：**
- carry = sum / 10
- digit = sum % 10
- 循环条件包含 carry!=0 处理最高位进位

**执行步骤：**
1. 逐位相加+进位
1. 创建新节点存储当前位
1. 更新进位
1. 遍历完链表后处理剩余进位


## ⚠️ 易错点

- 循环条件不能只写 `while l1 and l2`，因为两个链表长度不同，短的走完后长的那部分还要继续处理。
- 最后如果 `carry` 不为 0，要额外挂一个进位节点，不能丢掉。比如 `9 + 1 = 10`，结果链表要多一位。
- 新链表的头节点最好用哨兵节点（dummy），否则第一次插入时要特殊处理。

- 忘记 carry!=0 条件导致最高位进位丢失

- dummy 节点简化代码

## 📊 复杂度分析

- 时间复杂度：`O(max(m, n))`。`m` 和 `n` 是两个链表的长度，最多遍历较长链表的长度加一。
- 空间复杂度：`O(max(m, n))`。结果链表的长度最多是 `max(m, n) + 1`。
