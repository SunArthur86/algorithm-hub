> 难度：**Medium** | 分类：回溯 | 标签：数组, 字符串, 回溯, 矩阵

> 🔗 力扣链接：[https://leetcode.cn/problems/word-search/](https://leetcode.cn/problems/word-search/)

## 📝 题目描述

给你一个 `m × n` 的字母网格和一个目标单词 `word`，判断网格中是否存在一条连续路径，使得路径上的字母依次拼成 `word`。路径只能上下左右移动，不能斜着走，每个格子只能用一次。

- 题号：79
- 题目名：单词搜索
- 难度：中等
- 题型：回溯
- 为什么选这题：它把回溯从一维数组搬到了二维网格，加入了方向探索和访问标记，是理解 DFS+回溯的经典桥梁。

## 🧠 思考路径

先找到所有和 `word[0]` 匹配的起始格子，从每个起始格子出发做 DFS。每一步检查四个方向，如果相邻格子的字母匹配下一个字符，就继续深入。为了防止重复使用同一个格子，在访问后临时标记为已访问（比如改成 `#`），递归返回后再恢复。

这是一个典型的"尝试-标记-递归-回溯-恢复"流程。匹配成功就立即返回 `true`，不需要继续搜索。

## 💻 Java 题解

### 解法 1：DFS + 回溯

遍历每个格子作为起点，DFS匹配单词。用visited或原地修改标记已访问。

- ⏱ 时间复杂度：`O(m×n×3^L)`
- 💾 空间复杂度：`O(L)`

```java
class Solution {
    public boolean exist(char[][] board, String word) {
        for (int i = 0; i < board.length; i++)
            for (int j = 0; j < board[0].length; j++)
                if (dfs(board, word, 0, i, j)) return true;
        return false;
    }
    private boolean dfs(char[][] board, String word, int idx, int i, int j) {
        if (idx == word.length()) return true;
        if (i < 0 || i >= board.length || j < 0 || j >= board[0].length || board[i][j] != word.charAt(idx))
            return false;
        char temp = board[i][j];
        board[i][j] = '#'; // 标记
        boolean found = dfs(board, word, idx+1, i+1, j) || dfs(board, word, idx+1, i-1, j)
                     || dfs(board, word, idx+1, i, j+1) || dfs(board, word, idx+1, i, j-1);
        board[i][j] = temp; // 恢复
        return found;
    }
}
```

**关键点：**
- 原地修改board标记访问
- 四个方向DFS
- 匹配失败恢复board

**执行步骤：**
1. 每个格子作为起点尝试
1. 匹配第一个字符
1. DFS四个方向
1. 回溯恢复


## ⚠️ 易错点

- 标记和恢复必须成对出现，否则后续搜索会受影响。
- 边界检查要在访问数组之前做，否则会越界。
- 可以加一个优化：统计 `board` 和 `word` 的字符频率，如果 `board` 中某字符不够，直接返回 `false`。或者从字符频率更少的方向开始搜索（正序 vs 倒序）。

- board[i][j]='#' 要恢复，否则影响其他路径

- 先检查board[i][j]!=word[idx]

## 📊 复杂度分析

- 时间复杂度：`O(m × n × 4^L)`，其中 `L` 是 `word` 的长度。每个格子可能作为起点，每次 DFS 最多 4 个方向，深度为 `L`。
- 空间复杂度：`O(L)`，递归栈深度最多 `L`。
