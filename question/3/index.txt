e:T21c7,# 3. 无重复字符的最长子串

> 难度：**Medium** | 分类：滑动窗口 | 标签：哈希表, 字符串, 滑动窗口

> 🔗 力扣链接：[https://leetcode.cn/problems/longest-substring-without-repeating-characters/](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

## 📝 题目描述

给你一个字符串 `s`，找出里面最长的一段连续子串，要求这段子串里没有任何重复字符。返回这段子串的长度。

比如 `"abcabcbb"`，最长的无重复子串是 `"abc"`，长度是 3。

- 题号：3
- 题目名：无重复字符的最长子串
- 难度：中等
- 题型：滑动窗口
- 为什么选这题：它是滑动窗口的入门经典题，训练"右扩左缩"的窗口维护思维。

## 🧠 思考路径

朴素想法是枚举所有子串，检查每个子串里有没有重复字符，取最长的那个。但子串总数是 `O(n²)` 级别，每个检查又要 `O(n)`，总复杂度 `O(n³)`，显然太慢。

仔细想想，当我们在字符串上维护一个"窗口"时，窗口不断向右扩大，如果新加入的字符导致重复了，我们不需要从头再来，只需要把窗口左边界往右推，把重复的那个字符踢出去就行。这就是滑动窗口的核心思想。

用一个哈希表（或数组）记录窗口内每个字符的最新位置。右指针每走一步，如果当前字符已经在窗口内，就把左指针跳到它上次出现位置的下一个。

## 💻 Java 题解

### 解法 1：滑动窗口 + HashSet

左右指针维护无重复窗口。右指针尝试扩展，若遇到重复则左指针收缩直到无重复。记录窗口最大长度。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(min(m,n))`

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        Set<Character> set = new HashSet<>();
        int left = 0, maxLen = 0;
        for (int right = 0; right < s.length(); right++) {
            while (set.contains(s.charAt(right))) {
                set.remove(s.charAt(left));
                left++;
            }
            set.add(s.charAt(right));
            maxLen = Math.max(maxLen, right - left + 1);
        }
        return maxLen;
    }
}
```

**关键点：**
- HashSet 维护窗口内字符
- 遇到重复时左指针收缩
- 窗口大小 = right - left + 1

**执行步骤：**
1. left=0, 遍历 right
1. 若 s[right] 在 set 中，移除 s[left] 并 left++
1. 将 s[right] 加入 set
1. 更新最大窗口长度


### 解法 2：滑动窗口 + HashMap（优化跳转）

用 HashMap 记录字符最新位置，遇到重复时直接跳到重复字符的下一个位置，避免逐步收缩。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(min(m,n))`

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        Map<Character, Integer> map = new HashMap<>();
        int left = 0, maxLen = 0;
        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);
            if (map.containsKey(c)) {
                left = Math.max(left, map.get(c) + 1);
            }
            map.put(c, right);
            maxLen = Math.max(maxLen, right - left + 1);
        }
        return maxLen;
    }
}
```

**关键点：**
- HashMap 直接记录位置
- left = Math.max(left, map.get(c)+1)
- 避免重复字符在 left 之前的情况

**执行步骤：**
1. 遍历 right
1. 若字符在 map 中，left 跳到 max(left, 重复位置+1)
1. 更新 map 和最大长度


## ⚠️ 易错点

- 左边界不能往回跳。判断条件必须加上 `charIndex[ch] >= left`，否则已经不在窗口内的旧记录会把左边界错误地拉回去。
- 窗口长度是 `right - left + 1`，别忘了加 1（因为是闭区间）。
- 空字符串要返回 0，不是 1。

- left 跳转时用 Math.max 防止回退

- HashSet 方案中 while 不是 if

## 📊 复杂度分析

- 时间复杂度：`O(n)`。左右指针各最多遍历字符串一次。
- 空间复杂度：`O(min(m, n))`。`m` 是字符集大小（ASCII 为 128），`n` 是字符串长度。
