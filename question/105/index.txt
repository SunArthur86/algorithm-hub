e:T1eda,# 105. 从前序与中序遍历序列构造二叉树

> 难度：**Medium** | 分类：二叉树 | 标签：树, 数组, 哈希表, 分治, 二叉树

> 🔗 力扣链接：[https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)

## 📝 题目描述

给你一棵二叉树的前序遍历和中序遍历结果（都不含重复元素），请你构造并返回这棵二叉树。

前序遍历：根 → 左 → 右。第一个元素一定是根。
中序遍历：左 → 根 → 右。根元素左边的全是左子树，右边的全是右子树。

- 题号：105
- 题目名：从前序与中序遍历序列构造二叉树
- 难度：中等
- 题型：二叉树
- 为什么选这题：它是"遍历序列 ↔ 树结构"互转的经典题，深度考察对前序和中序遍历本质的理解。

## 🧠 思考路径

前序数组的第一个元素就是根节点。用这个根节点的值去中序数组里找位置，就能把中序数组分成三部分：

1. 根左边：左子树的中序遍历
2. 根本身
3. 根右边：右子树的中序遍历

左子树的大小（节点数）等于中序数组中根左边的元素个数。有了大小，就能在前序数组中划分出左子树的前序和右子树的前序。

然后递归处理左子树和右子树即可。

优化：用哈希表预处理中序数组中每个值的位置，避免每次线性查找根的位置。

## 💻 Java 题解

### 解法 1：递归 + HashMap

用 HashMap 存中序值到索引的映射。前序第一个是根，中序中定位根划分左右子树，递归构建。

- ⏱ 时间复杂度：`O(n)`
- 💾 空间复杂度：`O(n)`

```java
class Solution {
    int preIdx = 0;
    Map<Integer, Integer> inMap;
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        inMap = new HashMap<>();
        for (int i = 0; i < inorder.length; i++) inMap.put(inorder[i], i);
        return build(preorder, 0, inorder.length - 1);
    }
    private TreeNode build(int[] pre, int inLeft, int inRight) {
        if (inLeft > inRight) return null;
        TreeNode root = new TreeNode(pre[preIdx++]);
        int inMid = inMap.get(root.val);
        root.left = build(pre, inLeft, inMid - 1);
        root.right = build(pre, inMid + 1, inRight);
        return root;
    }
}
```

**关键点：**
- 前序第一个是根
- HashMap存中序索引→O(1)查找
- preIdx 全局递增

**执行步骤：**
1. 前序取根节点
1. 中序定位根划分左右
1. 递归构建左子树
1. 递归构建右子树


## ⚠️ 易错点

- `preIdx` 的递增必须在创建节点后立即执行，不能延迟。因为前序数组的顺序就是"根 → 左 → 右"，`preIdx` 是全局递增的。
- 递归顺序必须是先左后右。前序遍历中，根后面紧跟着的是左子树的所有节点，然后才是右子树。
- 中序数组的划分用 `[inLeft, rootPos-1]` 和 `[rootPos+1, inRight]`，根本身不包含在子树中。
- 题目保证不含重复元素，所以哈希表的映射是一一对应的。

- preIdx 必须在递归左子树之前递增

- 递归构建顺序：先左后右

## 📊 复杂度分析

- 时间复杂度：`O(n)`。每个节点处理一次，哈希查找 `O(1)`。
- 空间复杂度：`O(n)`。哈希表 `O(n)`，递归栈 `O(h)`。
