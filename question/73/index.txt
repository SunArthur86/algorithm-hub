e:T1b3b,# 73. 矩阵置零

> 难度：**Medium** | 分类：矩阵 | 标签：数组, 哈希表, 矩阵

> 🔗 力扣链接：[https://leetcode.cn/problems/set-matrix-zeroes/](https://leetcode.cn/problems/set-matrix-zeroes/)

## 📝 题目描述

给你一个 `m × n` 的矩阵，如果某个元素是 `0`，就把它所在的行和列全部设为 `0`。要求原地操作，不能用额外矩阵来存结果。

- 题号：73
- 题目名：矩阵置零
- 难度：中等
- 题型：矩阵
- 为什么选这题：它考察的是如何在有限空间内做标记，是"原地修改矩阵"思路的典型代表。

## 🧠 思考路径

最朴素的做法是先扫描一遍，把所有 `0` 的坐标记下来，再统一置零。这需要 `O(m+n)` 额外空间。

能不能做得更省？关键是"在哪做标记"。既然某行或某列要被置零，我们可以在矩阵的第一行和第一列做标记——用 `matrix[i][0]` 标记第 `i` 行是否要清零，`matrix[0][j]` 标记第 `j` 列是否要清零。但第一行和第一列本身也可能需要清零，所以要用两个额外变量单独记录它们。

这样就把额外空间压到了 `O(1)`。

## 💻 Java 题解

### 解法 1：原地标记（O(1)空间）

用第一行/列记录是否有零。先检查第一行列自身是否有零，然后用它们标记其他行列，最后根据标记置零。

- ⏱ 时间复杂度：`O(m×n)`
- 💾 空间复杂度：`O(1)`

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        int m = matrix.length, n = matrix[0].length;
        boolean firstRowZero = false, firstColZero = false;
        for (int j = 0; j < n; j++) if (matrix[0][j] == 0) firstRowZero = true;
        for (int i = 0; i < m; i++) if (matrix[i][0] == 0) firstColZero = true;
        for (int i = 1; i < m; i++)
            for (int j = 1; j < n; j++)
                if (matrix[i][j] == 0) { matrix[i][0] = 0; matrix[0][j] = 0; }
        for (int i = 1; i < m; i++)
            for (int j = 1; j < n; j++)
                if (matrix[i][0] == 0 || matrix[0][j] == 0) matrix[i][j] = 0;
        if (firstRowZero) for (int j = 0; j < n; j++) matrix[0][j] = 0;
        if (firstColZero) for (int i = 0; i < m; i++) matrix[i][0] = 0;
    }
}
```

**关键点：**
- 用第一行/列做标记数组
- 先记录第一行列自身是否有零
- 最后处理第一行列

**执行步骤：**
1. 检查第一行列是否有零
1. 用第一行列标记其他位置
1. 根据标记置零
1. 最后处理第一行列


## ⚠️ 易错点

- 必须先记录第一行和第一列的状态，再做标记。否则标记会覆盖掉原始信息。
- 置零的顺序：先处理内部区域（第 2 行/列开始），再处理第一行和第一列。
- 不能边扫描边置零，否则后面扫到的 `0` 可能是你自己设的，会连锁反应。

- 必须先记录第一行列的状态再标记

- 最后处理第一行列防止覆盖标记

## 📊 复杂度分析

- 时间复杂度：`O(m × n)`，遍历矩阵若干遍，每遍都是 `O(m × n)`。
- 空间复杂度：`O(1)`，只用了两个额外布尔变量。
