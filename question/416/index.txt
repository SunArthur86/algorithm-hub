e:T1596,# 416. 分割等和子集

> 难度：**Medium** | 分类：动态规划 | 标签：数组, 动态规划

> 🔗 力扣链接：[https://leetcode.cn/problems/partition-equal-subset-sum/](https://leetcode.cn/problems/partition-equal-subset-sum/)

## 📝 题目描述

给你一个只包含正整数的数组 `nums`，判断能否把它分成两个子集，使得两个子集的元素之和相等。

比如 `[1,5,11,5]` 可以分成 `[1,5,5]` 和 `[11]`，和都是 11，返回 `true`。

- 题号：416
- 题目名：分割等和子集
- 难度：中等
- 题型：动态规划
- 为什么选这题：它是 0-1 背包问题的经典变形——从"最大价值"变成"能否恰好装满"，理解了这个变形，背包问题就通了。

## 🧠 思考路径

首先，如果总和是奇数，不可能平分，直接返回 `false`。目标是判断能否从数组中选出若干元素，使它们的和恰好等于 `sum / 2`。

这就是 0-1 背包问题：容量为 `sum/2` 的背包，每个物品（数组元素）选或不选，问能否恰好装满。

定义 `dp[j]` 表示"能否凑出和为 `j`"。转移方程：`dp[j] = dp[j] || dp[j - nums[i]]`。从大到小更新 `j`，避免重复使用同一个元素。

初始化 `dp[0] = true`（和为 0 总是能做到，什么都不选）。

## 💻 Java 题解

### 解法 1：0-1背包（布尔dp）

target=sum/2。dp[j]表示能否凑成和j。倒序遍历防重复使用。

- ⏱ 时间复杂度：`O(n×target)`
- 💾 空间复杂度：`O(target)`

```java
class Solution {
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for (int n : nums) sum += n;
        if (sum % 2 != 0) return false;
        int target = sum / 2;
        boolean[] dp = new boolean[target + 1];
        dp[0] = true;
        for (int num : nums)
            for (int j = target; j >= num; j--)
                dp[j] = dp[j] || dp[j - num];
        return dp[target];
    }
}
```

**关键点：**
- 总和为奇数直接false
- 0-1背包倒序遍历
- dp[j]=dp[j]||dp[j-num]

**执行步骤：**
1. 计算总和判断奇偶
1. 初始化dp
1. 倒序背包
1. 返回dp[target]


## ⚠️ 易错点

- 内层循环必须倒序（从 `target` 到 `num`），否则同一个元素可能被重复使用（变成完全背包）。
- `dp[0] = true` 是关键初始化，表示"不选任何元素时和为 0"。
- 如果数组中有元素大于 `target`，该元素不可能被选中，但代码本身能处理。

- 倒序遍历防止重复使用同一元素

- 总和为奇直接返回false

## 📊 复杂度分析

- 时间复杂度：`O(n × target)`。`n` 为数组长度，`target = sum/2`。
- 空间复杂度：`O(target)`。一维 DP 数组。
