## ✨ 特性

### 📋 100 题完整题解（严格对齐热题 100）

## 📂 项目结构
```
├── index.html              # 主页面
├── manifest.json           # PWA 配置
├── sw.js                   # Service Worker
├── css/                    # 样式文件 (5个)
├── js/
│   ├── safe-ls.js          # localStorage 安全封装
│   ├── app.js              # 主逻辑 + 题解渲染 + 扩展区渲染
│   ├── problems-data.js    # 100 题数据（驱动 UI）
│   ├── sol-v2-*.js         # 13 批次 Java 题解（主列表 100 题）
│   ├── extra-solutions.js  # 🎁 扩展练习题解（4 题）
│   ├── diagrams.js         # 🖼 13 幅内联 SVG 静态图解
│   ├── algorithm-visualizer.js  # 基础动画
│   ├── visualizer-extended.js   # 扩展动画
│   ├── viz-engine.js       # 逐题动画 Canvas 渲染器
│   └── viz-traces-*.js     # 100 题动画轨迹 (6 文件)
└── data/
    └── hot100-raw.json     # 原始数据（与 leetcode.cn 热题 100 对齐）
```
